Perform the nearest neighbor 2-approximation algorithm to find a sub-tour for a given cluster.

This function performs the nearest neighbor 2-approximation algorithm to find a sub-tour for a given cluster. It starts from the first node in the cluster and iteratively finds the nearest unvisited neighbor, adding it to the tour. The time complexity is O(n), where n is the number of nodes in the cluster.

Parameters
clusterThe vector of indices representing the nodes in the cluster.

A vector representing the sub-tour for the cluster.
