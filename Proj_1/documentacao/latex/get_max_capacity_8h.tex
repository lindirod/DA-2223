Calculates the maximum capacity (or maximum flow) between two stations in a flow network. The time complexity of this function that uses common algorithms such as the Edmonds-\/\+Karp algorithm, is typically on the order of O(\+V $\ast$ E$^\wedge$2), where V is the number of vertices (stations in this case) and E is the number of edges (segments in this case) in the flow network.
